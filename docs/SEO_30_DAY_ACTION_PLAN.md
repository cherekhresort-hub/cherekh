# Cherekh Center SEO 30-Day Action Plan

Goal: improve rankings for high-intent searches around Thanchi/Bandarban resorts and bookings.

## Primary SEO Targets

- **Commercial:** `best resort in thanchi`, `thanchi resort booking`, `bandarban resort`
- **Category:** `family resort bandarban`, `couple resort bandarban`, `conference resort bandarban`
- **Informational:** `things to do in thanchi`, `best time to visit bandarban`, `thanchi travel guide`
- **Brand:** `cherekh center`, `cherekh center thanchi`

## Week 1: Technical + Tracking Foundation

1. **Search Console + GA4**
   - Verify domain property in Google Search Console.
   - Submit sitemap: `https://cherekhresort.com/sitemap.xml`.
   - In GA4, enable enhanced measurement and connect Search Console.
2. **Indexing checks**
   - Inspect homepage, rooms, conference, dining, experiences, contact in Search Console URL inspection.
   - Request indexing for any key page not yet indexed.
3. **Robots + redirects**
   - Ensure SPA fallback exists on Netlify (`/* /index.html 200`).
   - Confirm `robots.txt` allows crawling and references sitemap.
4. **Core Web Vitals**
   - Run PageSpeed on homepage + rooms page (mobile first).
   - Fix largest issues only (hero image size, JS payload, CLS from dynamic elements).

## Week 2: Landing Pages That Can Rank

Create or improve these pages with unique intent and copy (700-1200 words each):

1. `Best Resort in Thanchi` landing page
2. `Family Stay in Bandarban` landing page
3. `Couple Retreat in Bandarban` landing page
4. `Conference/Event Retreat in Bandarban` landing page

Each page must include:
- One clear keyword focus in title + H1 + first paragraph.
- Real differentiators (view, room types, food, access, activities, pricing range).
- Internal links to booking, rooms, contact.
- FAQ section (3-5 Q&A) with FAQ schema where relevant.

## Week 3: Authority + Local SEO

1. **Google Business Profile**
   - Fully optimize profile: services, amenities, categories, booking link, photos.
   - Add weekly photo updates.
2. **Reviews**
   - Collect at least 10-15 fresh Google reviews this month.
   - Ask guests to mention terms naturally (service, location, room type).
3. **Citations**
   - Ensure consistent NAP across Facebook, TripAdvisor, local directories, travel portals.
4. **Backlinks**
   - Outreach target: 8-12 relevant links from travel blogs/pages.
   - Offer itinerary posts, local guides, or creator stays for coverage.

## Week 4: Content + CTR Optimization

Publish 4 blog posts targeting informational demand:

1. `Top things to do in Thanchi and Bandarban`
2. `2-day and 3-day Thanchi itinerary`
3. `How to choose a resort in Bandarban`
4. `Best season to visit Bandarban hills`

For each:
- Add strong title tag (55-60 chars), compelling meta description (140-160 chars).
- Link to relevant booking/room pages.
- Add 2-3 original images with descriptive alt text.

Then optimize pages with impressions but low CTR:
- Rewrite title/meta to include value + differentiator.
- Add year/benefit qualifiers where useful (for example, "2026", "family-friendly", "river-view").

## On-Page Template (Use Across New Pages)

- **Title tag:** keyword + location + value prop
- **H1:** exact intent phrase
- **H2 sections:** rooms, amenities, pricing context, how to reach, FAQs
- **Schema:** Resort/LodgingBusiness + FAQ + Breadcrumb
- **Internal links:** minimum 5 contextual links
- **Media:** compressed images, lazy loading, descriptive alt tags

## Weekly KPI Dashboard

Track every Monday:

- Impressions (Search Console)
- Clicks + CTR
- Average position for top 20 queries
- Number of indexed pages
- Organic sessions (GA4)
- Booking leads from organic traffic

## 90-Day Expectation

If execution is consistent:
- Branded keywords: strong top visibility
- Long-tail local queries: realistic page 1 movement
- Competitive head terms (`best resort in thanchi`): meaningful gains, but #1 depends on backlinks, reviews, and competitors

## Priority Order (if time is limited)

1. Search Console + indexing + Core Web Vitals
2. 3 high-intent landing pages
3. Google reviews + local citations
4. Backlink outreach
5. Blog cadence and CTR tuning
